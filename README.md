## Installation

```bash
    npm i @dao-vue/alert    # using npm
    yarn add @dao-ve/alert  # using yarn
```
## Adding to project
```html
import Vue from 'vue';
import DaoAlert from '@dao-vue/alert'

Vue.use(DaoAlert);
```

## Usage

```html
<dao-alert message="My alert" />
<dao-alert :close="true" >My Alert</dao-alert>
```
